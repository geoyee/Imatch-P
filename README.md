# Imatch-P
A demo using SuperGlue and SuperPoint to do the image matching task based PaddlePaddle.

![下载](https://user-images.githubusercontent.com/71769312/158503692-1ffb04fa-47f5-4517-b4b6-aaea6437db97.png)

## How to use
``` shell
cd Imatch-P/
python demo.py
```
